## 1. Project Overview
**Empowering the Nation** is an educational organization offering short- and long-term community training courses such as **First Aid, Sewing, Landscaping, Life Skills, Child Minding, Cooking**, and **Garden Maintenance**.  
This website was developed to provide an accessible digital platform where users can explore courses, view details, calculate fees, and contact the organization.

The web design mirrors the brand identity of Empowering the Nation, featuring a **clean, modern layout** with **navy blue, forest green, and cream** tones reflecting trust, growth, and inclusivity.

---

## 2. Features and Functionality
- **Home Page:** Introduction to the organization, mission, and course highlights.  
- **Courses Page:** Lists both Six-Week and Six-Month programs with summaries and links.  
- **Course Details:** Displays purpose, fees, and detailed content for each course.  
- **Fee Calculator:** Allows users to select multiple courses and view total cost dynamically.  
- **Contact Page:** Contains a functional enquiry form and integrated Google Map location.  
- **Responsive Layout:** Website automatically adjusts to desktop, tablet, and mobile views.

---

## 3. Technologies Used
- **HTML5** – Structure and content markup  
- **CSS3** – Styling, layout, and responsive design  
- **JavaScript (ES6)** – Interactive elements (fee calculator and contact form validation)  
- **Bootstrap 5** – Grid layout and responsive navigation  
- **Font Awesome** – Icons for user interface enhancement  
- **Google Fonts** – Typography customization
